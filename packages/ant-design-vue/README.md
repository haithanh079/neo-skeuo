# @neo-skeuo/ant-design-vue

[![npm version](https://img.shields.io/npm/v/@neo-skeuo/ant-design-vue.svg)](https://www.npmjs.com/package/@neo-skeuo/ant-design-vue)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](../../LICENSE)

Ant Design Vue 4+ provider with neo-skeuo tokens applied via `ConfigProvider`. **Alpha** maturity.

```bash
npm install @neo-skeuo/ant-design-vue @neo-skeuo/web-css ant-design-vue vue
```

```vue
<script setup>
import { NeoSkeuoAntDesignVueProvider } from "@neo-skeuo/ant-design-vue";
</script>

<template>
  <NeoSkeuoAntDesignVueProvider theme="light">
  </NeoSkeuoAntDesignVueProvider>
</template>
```

Requires `ant-design-vue` and `@neo-skeuo/web-css` in your app.
