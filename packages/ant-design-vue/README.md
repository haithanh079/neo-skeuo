# @neo-skeuo/ant-design-vue

Ant Design Vue 4+ provider with neo-skeuo tokens applied via `ConfigProvider`.

```vue
<script setup>
import { NeoSkeuoAntDesignVueProvider } from "@neo-skeuo/ant-design-vue";
</script>

<template>
  <NeoSkeuoAntDesignVueProvider theme="light">
  </NeoSkeuoAntDesignVueProvider>
</template>
```

Requires `ant-design-vue` and `@neo-skeuo/web-css` in your app.
