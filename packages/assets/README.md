# @neo-skeuo/assets

[![npm version](https://img.shields.io/npm/v/@neo-skeuo/assets.svg)](https://www.npmjs.com/package/@neo-skeuo/assets)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](../../LICENSE)

SVG assets for neo-skeuo borders and paper textures.

```bash
npm install @neo-skeuo/assets
```

```ts
import { borderRaised, paperTexture, assetPaths } from "@neo-skeuo/assets";
```

Subpath imports: `@neo-skeuo/assets/border-raised.svg`, etc. Used by `@neo-skeuo/web-css` token variables.
